**关键技术决策**：

**`lut.ts` 测试用例覆盖范围**：

#### 正常场景
- `clamp(v, min, max)`：边界值 0/1、超出范围 clamp、默认参数
- `parseCube(content, id, name)`：
  - 标准 3D LUT 文本解析（含 TITLE/LUT_3D_SIZE/数据行）
  - 注释行 `#` 跳过
  - 空行跳过
  - TITLE 提取带引号内容
- `sampleLut3D(lut, r, g, b)`：
  - 边界采样 (0,0,0) (1,1,1)
  - 中心采样 (0.5, 0.5, 0.5)
  - 三线性插值精度验证
- `applyLut3D(imageData, lut)`：
  - identity LUT 不改变像素
  - 非法 imageData 容错
- `createBuiltInLut(id, name, size, fn)`：
  - 生成 size³ × 3 长度的 Float32Array
  - fn 被正确调用
- `builtInLuts`：
  - 5 个内置 LUT 全部存在（identity/teal-orange/warm-vintage/cool-drama/high-contrast）
  - 数据长度匹配 size³×3

#### 异常场景
- `parseCube`：size=0 返回 null、数据行数不匹配返回 null、含 NaN 行跳过、1D LUT 返回 null
- `sampleLut3D`：超出 [0,1] 范围的输入 clamp
- `applyLut3D`：空 imageData

**`imageProcessor.ts` 测试用例覆盖范围**：

#### 正常场景
- `rgbToHsl(r, g, b)`：
  - 纯红 (255,0,0) → (0, 1, 0.5)
  - 纯绿 (0,255,0) → (1/3, 1, 0.5)
  - 纯蓝 (0,0,255) → (2/3, 1, 0.5)
  - 白色 (255,255,255) → (0, 0, 1)
  - 黑色 (0,0,0) → (0, 0, 0)
  - 灰色 (128,128,128) → (0, 0, ~0.5)
- `hslToRgb(h, s, l)`：
  - 灰度（s=0）→ r=g=b=l
  - 与 rgbToHsl 互为逆运算（精度容差 1e-6）
- `buildCurveMap(points)`：
  - 恒等曲线 [(0,0),(1,1)] → map[i] = i
  - 反相曲线 [(0,1),(1,0)] → map[i] = 255 - i
  - 单点曲线（< 2 个点）
  - 乱序 points 自动排序
- `applyHSL(r, g, b, hsl)`：
  - 全 0 调整 → 不改变原色
  - 红色 hue 偏移
  - 范围因子计算（red/orange/yellow 等 12 个 key）
- `applyTemperature(r, g, b, temp, tint)`：
  - temp=0 tint=0 → 不变
  - 正温（暖）：r 增加，b 减少
  - 负温（冷）：r 减少，b 增加
  - tint 影响 g/b
- `applyVibrance(r, g, b, vibrance)`：
  - vibrance=0 → 不变
  - 高饱和度像素受影响小，低饱和度像素受影响大
- `applyDehaze(r, g, b, dehaze)`：
  - dehaze=0 → 不变
  - 正值去雾（提亮暗部、降亮部）
- `applySplitTone(r, g, b, lum, params)`：
  - 全 0 参数 → 不变
  - shadowHue/shadowSaturation 影响暗部
  - highlightHue/highlightSaturation 影响亮部
  - balance 调整权重
- `getWatermarkPosition(position, cw, ch, ww, wh, margin, customX, customY)`：
  - 9 种预设位置（topLeft/topCenter/.../bottomRight）
  - custom 使用 customX/customY
  - 默认 fallback 到 bottomRight
- `formatWatermarkText(content)`：
  - `{date}` 替换为当前日期 YYYY.MM.DD
  - `{time}` 替换为当前时间 HH:MM
  - 大小写不敏感（`{DATE}`/`{Date}`）
  - 无占位符原文返回

#### 异常场景
- `rgbToHsl`：超出 [0,255] 的输入（当前实现会除以 255，结果超出 [0,1]，固化此行为）
- `buildCurveMap`：空 points 数组（当前实现 fallback 到恒等）
- `applyHSL`：hsl 参数缺失某个 key（默认 {hue:0, saturation:0, lightness:0}）
- `getWatermarkPosition`：未知 position 字符串 fallback
- `formatWatermarkText`：空字符串、null（容错）

## 七、风险与应对
