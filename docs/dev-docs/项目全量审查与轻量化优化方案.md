### 1.2 整体评价

项目在功能完整度与健壮性上已达生产级，历史 P0 修复扎实（单实例锁、异常捕获、退出清理、路径白名单、操作历史等均已实现）。本轮审查聚焦三大系统性问题：

1. **主进程入口失控膨胀**：`src/main/index.ts` 单文件 4051 行，承载 60+ IPC 注册 + 协议处理 + 扫描 + 修复 + 快捷方式等十余种职责，所有后续维护成本被放大。
2. **token 化不彻底**：24+ 文件硬编码颜色/阴影/模糊半径，主题切换时部分组件不跟随；紧凑模式仅覆盖 3 个类，承诺未兑现。
3. **渲染进程内存管理薄弱**：模块级缓存无淘汰策略（useGameParams）、大文件全量累积（preview OOM）、imageProcessor 峰值 448MB。

- **位置**：`src/main/database/connection.ts` 全部方法、`src/main/index.ts` 所有 `db.prepare().all()/run()`
