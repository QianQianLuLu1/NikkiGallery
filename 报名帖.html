创意名称 + 创意介绍
暖暖相册
Infinity Nikki Gallery Manager
想解决什么问题
《无限暖暖》是一款以穿搭摄影为核心玩法的开放世界游戏，玩家在游戏中会积累大量截图和照片，但这些媒体文件散落在游戏安装目录的多个子文件夹中，没有任何官方管理工具。玩家想找某张特定截图只能在文件夹里手动翻找；想修图美化需要导入第三方软件；想分享到社交平台还要经历复制、粘贴、传输的繁琐流程。暖暖相册就是为了解决"截图多、找图难、修图烦、分享慢"这个痛点而生的。
为什么会想到做这个
作为《无限暖暖》的活跃玩家，我自己就有上万张游戏截图，每次想找一张特定照片都要翻很久。游戏自带的相册功能非常简陋，只能按时间浏览，没有分类、搜索、编辑功能。玩家社区里几乎每天都有人问"怎么批量导出截图""怎么给游戏截图加滤镜""怎么按套装找图"，这个需求一直存在但一直没人解决。
大概是什么产品
一款 Windows 桌面应用（Electron + React + TypeScript），集智能全盘扫描、六维度智能分类、专业修图、重复检测、一键分享于一体的游戏截图一站式管理工具。
图库浏览 — 自动扫描、网格展示、多维度筛选
专业修图 — 12 项参数 + HSL + 曲线 + LUT
风格滤镜 — 8 款一键预设
Electron 28 React 18 TypeScript 5.3 Tailwind CSS Zustand better-sqlite3 sharp ffmpeg i18next

目标用户及痛点
面向哪些用户
《无限暖暖》的活跃玩家，尤其是热爱游戏内摄影和穿搭分享的"暖暖摄影师"。他们频繁截图、注重图片质量、有修图和社交分享需求，但对专业修图软件和文件管理工具的使用门槛较高。游戏全球注册用户数千万，其中占比可观的摄影爱好者都是潜在用户。
在什么场景下使用
游戏结束后整理当天拍摄的照片，按场景、套装、时段智能分类归档
从上万张截图中快速找到特定套装或场景的照片，准备发社交平台分享
对截图进行调色、加滤镜、加水印等后期处理，无需打开 Photoshop
检测并清理重复和相似截图，释放磁盘空间
将精选照片批量导出并一键分享到微信、QQ 等社交应用
当前痛点
01
截图散落在游戏目录的 22 个子文件夹中，玩家根本不知道照片存在哪，只能用 Windows 文件管理器逐个翻找
02
没有缩略图预览、没有分类筛选、没有搜索功能，找一张特定截图可能要翻几十分钟
03
想修图要导入 Photoshop 等第三方专业软件，对非技术背景的玩家操作门槛太高
04
分享截图需要手动复制文件、切换应用、粘贴发送，步骤繁琐且不支持批量操作
05
多账号玩家的截图混在一起无法按角色区分，重复截图占用大量磁盘空间却无法有效清理

价值与意义

效率提升

暖暖相册将截图管理从"手动翻文件夹"升级为智能一站式管理。全盘自动扫描 5 万文件 30 秒完成，六维度智能分类让找图从分钟级降到秒级，内置 12 项参数专业修图工具省去第三方软件切换成本，一键分享将多步操作压缩为单次点击。对每一位热爱游戏摄影的玩家来说，这意味着更多时间用于创作，更少时间消耗在文件管理上。

社会价值

《无限暖暖》的玩家群体中有大量非技术背景的女性用户和年轻用户，她们对"整理文件夹""使用专业修图软件"有天然门槛。暖暖相册通过极简交互和自动化流程，降低了数字资产管理的门槛，让每一位玩家都能轻松管理自己的游戏回忆。工具内置的水印功能也帮助创作者保护原创作品，促进游戏摄影社区的健康发展。
