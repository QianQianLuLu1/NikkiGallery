# Nikki Gallery 2.1 正式发布：专为无限暖暖打造的游戏影像管理工具

还在层层嵌套的游戏目录里翻找截图、旅行手账和打卡照片？还在为整理海量游戏影像手动分类、逐个重命名？专为《无限暖暖》打造的专属桌面图库管理工具 **Nikki Gallery（暖暖图库）2.1 版本** 现已正式发布，一键搞定游戏照片的分类、整理、批量管理，让你专注记录搭配，不用再为找图烦恼。

## 精准适配：双服全兼容，开箱即用
作为《无限暖暖》专属工具，Nikki Gallery 2.1 原生支持**国服与国际服双版本**，可自动识别 Steam、Epic、官方启动器等多种渠道的游戏安装路径，无需手动指定文件夹，打开软件一键扫描即可导入所有游戏内影像。

本版本仅提供标准安装包形式，跟随引导完成安装后即可使用，适配 Windows 10 20H2 及以上、Windows 11 全系列 64 位系统。

## 核心能力：智能分类+批量管理，整理效率翻倍
### 1. 按游戏场景自动归档，告别乱序文件夹
软件内置智能路径匹配引擎，扫描时会根据文件所在目录自动识别影像来源，精准划分 5 大类游戏内容：
- 游戏截图（ScreenShot）
- 低清缩略图（NikkiPhotos_LowQuality）
- 旅行手账（MagazinePhotos）
- 世界巡游打卡照（ClockInPhoto）
- 趣拼海报原图（Collage_CollagePhoto）

所有分类自动完成，无需手动打标签；支持单分类查看、多分类组合筛选，搭配文件名、日期、分辨率排序，想找的照片一秒定位。

### 2. 全功能批量操作，管理一步到位
针对大量游戏影像的整理需求，软件提供完整的批量处理能力：
- 批量删除、复制、移动，一键清理冗余素材
- 批量重命名、格式转换、画质压缩，适配分享、存档不同需求
- 所有操作均支持自动备份与撤销，不用担心误删误改

### 3. 三套高颜值主题，适配不同审美
除了实用功能，Nikki Gallery 2.1 还内置三套完整视觉主题，可在设置中一键切换：
- **默认简约**：清爽浅色界面，适配系统原生风格
- **未来科技**：深空暗色+霓虹青蓝点缀，科技感拉满
- **柔粉轻奢**：奶白基底+低饱和烟粉材质，贴合游戏柔美调性

所有主题均采用高级毛玻璃材质与流畅过渡动效，兼顾质感与性能，不会占用额外系统资源。

## 轻量稳定：低占用，零卡顿
全异步多线程架构保证软件运行全程界面无阻塞，扫描上千张照片也不会卡顿；全链路异常容错机制，权限受限目录、损坏文件、特殊文件名均不会导致软件闪退。
后台空闲时内存占用极低，挂在后台不影响游戏运行，边玩边整理毫无压力。
